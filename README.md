## ⚡ Features

- **17 Full Lessons** covering the entire eJPTv2 curriculum from zero to exam-ready
- **Interactive Hacker Terminal** with 20+ simulated pentesting tools (nmap, metasploit, hydra, sqlmap...)
- **130+ Quiz Questions** — Multiple choice, drag-and-drop, type-the-command
- **Gamification** — XP system, 7 ranks (Script Kiddie → Shadow Admin), streaks, achievements
- **Sound Effects** — Audio feedback via Web Audio API (no files needed)
- **Progress Tracking** — Saved in localStorage, pick up where you left off
- **Dark Hacker Theme** — Glowing green/cyan aesthetic
- **Zero Dependencies** — Pure HTML/CSS/JS, no frameworks, no build tools
- **Mobile Responsive** — Works on any device

## 📚 Course Structure

| Section | Topics | Lessons |
|---------|--------|---------|
| **Prerequisites** | Networking, Web Apps, Pentesting Intro | 3 |
| **Assessment Methodologies** | OSINT, Scanning, Enumeration, Vuln Assessment | 4 |
| **Auditing Fundamentals** | CIA Triad, Compliance, Risk | 1 |
| **Host & Network Pentesting** | System Attacks, Network Attacks, Metasploit, Exploitation, Post-Exploitation, Social Engineering | 6 |
| **Web App Pentesting** | HTTP Protocol, Web Attacks (SQLi, XSS, LFI, CSRF, IDOR) | 3 |

## 🚀 Run Locally

```bash
git clone https://github.com/7mx238/ejptv2-course.git
cd ejptv2-course
python3 -m http.server 8080
# Open http://localhost:8080
```

## 🛠 Tech

- Vanilla JavaScript (ES Modules)
- CSS Custom Properties + Animations
- Web Audio API for sound effects
- localStorage for progress persistence
- Zero external dependencies
